**Metal Pipe**

Metal Pipe adalah game keberuntungan otomatis berdurasi singkat untuk 2 sampai 8 pemain. Game ini memang dirancang konyol dan tidak membutuhkan strategi apa pun setelah host memulai permainan: pipa akan memilih pemain untuk mengayun, memilih target, memainkan efek suara "bonk", dan mengumumkan pemenangnya.

Fokus utama game ini adalah kejelasan, bukan kerumitan. Karena semua hasilnya acak, game ini mengandalkan feedback audio yang cepat, pengumuman eliminasi yang jelas, serta kontrol status yang ramah bagi pengguna layar sentuh.

**Cara Bermain**

Saat permainan dimulai, semua pemain yang aktif ditandai sebagai "berdiri". Game kemudian akan menjadwalkan urutan "bonk" secara otomatis.

Untuk setiap "bonk":

* Pemain yang berdiri dipilih secara acak sebagai pembawa pipa.
* Target dipilih dari pemain yang masih berdiri.
* Jika opsi "self-bonk" aktif, pembawa pipa bisa saja secara tidak sengaja memukul diri sendiri.
* Target yang terpukul akan tereliminasi.

Tidak ada giliran manual selama urutan "bonk" berlangsung. Perintah **Giliran siapa** akan menjelaskan bahwa Metal Pipe berjalan secara otomatis.

**Mode Single Bonk**

Mode ini adalah pengaturan bawaan.

* Jika pembawa pipa memukul orang lain, pembawa pipa menang.
* Jika pembawa pipa memukul diri sendiri, semua pemain lainnya menang.

Mode ini paling cocok untuk guyonan singkat di meja.

**Mode Multiple Bonks**

Saat **Multiple bonks** diaktifkan, urutan akan terus berlanjut setelah setiap eliminasi. Pembawa pipa dan target baru akan dipilih hanya dari pemain yang masih berdiri. Permainan berakhir ketika hanya tersisa satu pemain, dan pemain tersebut menjadi pemenangnya.

Jika "self-bonk" diaktifkan dalam mode ini, seorang pemain bisa tereliminasi sendiri selama urutan berlangsung.

**Status dan Kontrol Sentuh**

Pemain dengan layar sentuh selalu bisa mengakses kontrol meja standar selama permainan berlangsung:

* **Lihat status pipa:** Membuka panel status langsung yang menampilkan mode, jumlah bonk, bonk terakhir, serta status berdiri/eliminasi setiap pemain.
* **Giliran siapa:** Menjelaskan bahwa game berjalan otomatis dan melaporkan berapa banyak pemain yang masih bertahan.
* **Siapa yang ada di meja:** Membacakan daftar pemain dan penonton yang sedang berada di meja.

**Opsi Game**

* **Multiple Bonks:** Bawaan: mati. Jika mati, satu bonk langsung menentukan hasil. Jika nyala, bonk akan berlanjut sampai hanya tersisa satu pemain.
* **Izinkan Self-Bonk:** Bawaan: nyala. Jika nyala, pembawa pipa bisa terpilih sebagai target. Jika mati, pembawa pipa selalu memukul orang lain.

Opsi-opsi ini tidak saling bertabrakan. Semua kombinasi bisa digunakan.

**Opsi Game Pribadi**

* **Pengumuman singkat:** Memperpendek pengumuman bonk dan pemenang namun tetap memberikan info jelas mengenai siapa pembawa pipa, targetnya, dan hasil eliminasinya.
* **Konfirmasi aksi berisiko:** Tidak digunakan. Metal Pipe tidak memiliki aksi gameplay manual yang berisiko setelah host memulai permainan.

**Pintasan Keyboard**

* **C:** Lihat status pipa.
* **T:** Cek giliran siapa atau fase otomatis.

Tidak ada aksi bonk manual. Begitu game dimulai, pipa yang akan menentukan hasilnya.
